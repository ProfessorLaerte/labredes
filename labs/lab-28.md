# Atividade Extraclasse  
## Tema 07 — Modelagem de Dados (Modelo Entidade-Relacionamento)

Disciplina: Bancos de Dados  

---

## Objetivo

Aplicar os conceitos de **entidade, atributo, relacionamento, cardinalidade e chave primária** na modelagem conceitual de um sistema simples.

---

## Enunciado

Uma universidade deseja informatizar o processo de **matrícula em disciplinas**.

Com base nesse cenário, responda às questões abaixo.

---

## Questões

1. Identifique **duas entidades principais** do sistema.

2. Liste **três atributos** para cada entidade identificada.

3. Descreva **o relacionamento existente entre essas entidades**.

4. Indique **a cardinalidade desse relacionamento**.

5. Sugira **uma possível chave primária** para cada entidade.

---

## Orientações

- Resposta individual.  
- Texto curto (aproximadamente meia página).  
- A atividade será discutida na próxima aula.

---

## Observação

O objetivo da atividade é desenvolver a **capacidade de abstração e organização lógica dos dados**, não sendo necessária pesquisa técnica aprofundada.
